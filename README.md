## Installation

clone the repo

```bash
  gh repo clone kyrregjerstad/social-media-client-workflow
```

navigate to the dir and install the dependencies
```bash
  cd social-media-client-workflow
  npm i
```

start the dev server:
```bash
npm run start
```
